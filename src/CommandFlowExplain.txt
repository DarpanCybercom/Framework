// How to Add Packages In System 

    - Add Packages In Global Level in System.

        npm install -g @darpanvadher2001/framework

    Explain : 
                This Packages is Use for Install Custom Command for use Custom framework as your Project.


How Command Use : 


    framework [options] [custom command] [Arguments]
 
    - framework   - [ Main Prefix Command ] 


---/--- Custom Command List ---/---

1) Initialization Project

        framework --init [Project Name]  |  framework -i [Project Name]

        Explain : 
            
            This Command is use for initialization of project in folder.

        If you want initialization in current folder then use this command 

            framework --init | framework -i 

        If you want initialization in another folder then use this command.

            framework --init FrameworkTest | framework -i FrameworkTest


2) Add New Module In Project 

        framework addModule [ModuleName]

        Explain : 

            This Command is use for add Module in Your Initialized Project.

3) Add New Model In Module in Project

        framework addModel [ModelName]

        Explain : 

            This Command is use for add Model in Module in Your Initialized Project.

3) Add File Upload Feature in API

        framework addFileUpload 

        Explain : 

            This Command is use for add File Upload Feature In API.


4) Add Authentication In API 

        framework addAuth

        Explain : 

            This Command is use for add Authentication in API with DataBase or Without DataBase.
